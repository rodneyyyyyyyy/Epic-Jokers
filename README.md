# Epic-Jokers
A mod for Balatro that adds 5 new Epic Jokers, that can be obtained through the "The Saul" Spectral card.
# Additions:
cano (Joker): This Joker gains X0.75 Mult when a face card is added to the deck\ntrib (Joker): This Joker gains X0.5 Mult when a played King or Queen is scored. Resets when any other rank is scored\nyork (Joker): This Joker gains X1 Mult every 15 cards scored\nchic (Joker): If poker hand is a Straight Flush, X0.5 blind requirement\nperk (Joker): When a consumable is used, 1 in 2 chance to create a Negative copy of a consumable in your possession of the same type\nThe Saul (Spectral Card): Creates an Epic Joker
